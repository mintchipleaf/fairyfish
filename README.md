Fairyfloss theme for <a href="https://github.com/fish-shell/fish-shell">Fish</a>, based on the <a href="https://github.com/sailorhg/fairyfloss">original</a> editor theme by sailorhg


Install with a plugin manager like <a href="https://github.com/jorgebucaran/fisher">Fisher</a> or manually by placing the .theme file under `~/.config/fish/themes/`

```sh
fisher install mintchipleaf/fairyfish
```
